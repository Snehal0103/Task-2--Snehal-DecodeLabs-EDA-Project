# DecodeLabs Exploratory Data Analysis Project

## Overview
This project focuses on performing Exploratory Data Analysis (EDA) on an e-commerce dataset using Python.

## Analysis Performed
- Statistical Summary
- Trend Analysis
- Revenue Analysis
- Product Analysis
- Payment Method Analysis

## Visualizations Created
- Monthly Sales Trend
- Order Status Distribution
- Top Products Analysis
- Revenue Analysis Charts

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Outcome
Generated meaningful business insights through professional data analysis and visualizations.
